CanAddContact

Does the Service support adding in a new contact from the on device contacts when creating a group.

True if the Service allows adding a new contact from the on device contacts when creating a group. False if the Service only allows selecting from a predefined set of contacts specified by the Service.

FastSearch

Does the Service support incremental searching.

True if you can submit search requests to the Service as the user is typing. False if you must submit a search request to the Service after the user has designated via some type of submit indication to search on the text they have input.

SearchHint

Specify the text string to be used as a hint in the New Message search edit box.

Plugin developers may want to use this string to guide users in what can be searched on - such as app-specific contact name and/or device-specific contact phone number.

If null is returned, will default to a language specific version for "Search".
