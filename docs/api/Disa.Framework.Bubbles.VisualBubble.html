IsServiceIdSequence

A flag to specify if the IdService field can be used to determine message ordering in the event that two VideoBubbles have the same Time value.

Defaults to false.

Discussion: The Time field has a granularity in seconds. If two VisualBubbles arrive at the same second, a Service may wish to specify that the IdService may be used to specify the sequence of the VisualBubbles.
